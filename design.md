# Дизайн Quiz Flashcards App

## Общая концепция

Минималистичный, современный дизайн с акцентом на читаемость и удобство использования.

## Цветовая схема

### Светлая тема
- Фон: `#F9FAFB` (gray-50)
- Карточки: `#FFFFFF` (white)
- Текст: `#111827` (gray-900)
- Акцент (правильно): `#10B981` (emerald-500)
- Акцент (неправильно): `#EF4444` (red-500)
- Границы: `#E5E7EB` (gray-200)

### Темная тема
- Фон: `#111827` (gray-900)
- Карточки: `#1F2937` (gray-800)
- Текст: `#F9FAFB` (white)
- Акцент (правильно): `#10B981` (emerald-500)
- Акцент (неправильно): `#EF4444` (red-500)
- Границы: `#374151` (gray-700)

## Компоненты

### 1. Header (Шапка)
- Фиксированная вверху страницы
- Логотип слева (иконка "Q" в градиентном квадрате)
- Навигация по режимам (Тест / Результаты / Админ)
- Переключатель темы справа
- Кнопка экспорта CSV (если есть вопросы)

**Визуальные характеристики:**
- Тень: `shadow-sm`
- Скругление: нет (прямоугольная)
- Высота: 64px (h-16)

### 2. Welcome Screen (Экран приветствия)
- Центрированный контент
- Заголовок: "Добро пожаловать в Quiz Flashcards!"
- Две карточки в ряд (на десктопе):
  - Локальный файл (слева)
  - Загрузка файла (справа)
- Информация о поддерживаемых форматах CSV внизу

**Карточки:**
- Скругление: `rounded-2xl`
- Тень: `shadow-lg`
- Отступ: `p-6`
- Граница: `border border-gray-200 dark:border-gray-700`

### 3. Quiz Card (Карточка вопроса)

**Структура:**
```
┌─────────────────────────────────────┐
│  Прогресс-бар (сверху)              │
│  ▓▓▓▓▓▓▓▓░░░░░░░░░░  Вопрос 3/20    │
├─────────────────────────────────────┤
│  Текст вопроса (крупный, жирный)     │
│                                       │
│  [A) Вариант 1] ← зеленая/красная    │
│  [B) Вариант 2]                      │
│  [C) Вариант 3]                      │
│  [D) Вариант 4]                      │
│                                       │
│  [Объяснение] (если есть)             │
├─────────────────────────────────────┤
│  [← Предыдущий]  [Время]  [Следующий →] │
└─────────────────────────────────────┘
```

**Варианты ответов:**
- По умолчанию: светло-серый фон, темный текст
- При наведении: более темный фон
- Правильный ответ: `bg-emerald-500` + белый текст + иконка CheckCircle2
- Неправильный ответ: `bg-red-500` + белый текст + иконка XCircle
- Правильный ответ (если выбран неправильный): тоже зеленый

**Анимации:**
- Плавное появление карточки (fade + slide)
- Масштабирование кнопок при наведении (scale 1.02)
- Пружинная анимация иконок (CheckCircle2, XCircle)

### 4. Results Screen (Экран результатов)

**Структура:**
```
┌─────────────────────────────────────┐
│          [Трофей иконка]             │
│         "Тест завершен!"             │
│             85%                      │
│   Правильных ответов: 17 из 20       │
├─────────────────────────────────────┤
│  [Правильно: 17] [Неправильно: 3]    │
│  [Время: 5м 23с]                     │
├─────────────────────────────────────┤
│  [Попробовать снова] [Экспорт]       │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  Детализация ответов                 │
│  ┌───────────────────────────────┐   │
│  │ ✓ 1. Вопрос 1                │   │
│  │    Выбранный: Правильный      │   │
│  │    Время: 12.3с               │   │
│  └───────────────────────────────┘   │
│  ┌───────────────────────────────┐   │
│  │ ✗ 2. Вопрос 2                │   │
│  │    Выбранный: Неправильный    │   │
│  │    Правильный: Правильный     │   │
│  │    Время: 8.5с                │   │
│  └───────────────────────────────┘   │
└─────────────────────────────────────┘
```

**Цвета результатов:**
- Процент >= 80%: зеленый (emerald)
- Процент >= 60%: желтый (yellow)
- Процент < 60%: красный (red)

### 5. Admin Panel (Админ-панель)

**Структура:**
```
┌─────────────────────────────────────┐
│  Админ панель                        │
│  [Загрузить CSV] [Экспорт CSV]       │
│  [Скачать шаблон] [Сбросить всё]    │
│  Всего вопросов: 20                  │
│  [+ Добавить вопрос]                 │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  1. Вопрос 1?                        │
│     A) Вариант 1                     │
│     B) Вариант 2 (правильный)        │
│     C) Вариант 3                     │
│     D) Вариант 4                     │
│     [Редактировать] [Удалить]        │
└─────────────────────────────────────┘
```

**Режим редактирования:**
- Inline редактирование
- Поля ввода для вопроса, вариантов, правильного ответа, объяснения
- Кнопки "Сохранить" и "Отмена"

## Типографика

- Заголовки: `font-bold`, размеры от `text-xl` до `text-4xl`
- Основной текст: `text-base` (16px)
- Мелкий текст: `text-sm` (14px)
- Шрифт: Inter, system-ui, sans-serif

## Отступы и размеры

- Контейнер: `max-w-4xl` или `max-w-6xl` (центрирован)
- Отступы страницы: `px-4 py-8`
- Отступы карточек: `p-6` или `p-8`
- Скругление карточек: `rounded-2xl` (16px)
- Скругление кнопок: `rounded-lg` (8px)

## Адаптивность

- Mobile: одинарная колонка, полная ширина
- Tablet: 2 колонки в некоторых местах
- Desktop: максимальная ширина контейнера, центрирование

## Иконки

Используются из библиотеки Lucide React:
- Home, BarChart3, Settings - навигация
- Moon, Sun - тема
- CheckCircle2, XCircle - правильный/неправильный ответ
- Clock - время
- Upload, Download, FileText - файлы
- Edit2, Save, X, Trash2, Plus - админ-панель
- RotateCcw - перезапуск
- Trophy - результаты

## Анимации

- Framer Motion для плавных переходов
- Анимация появления карточек: fade + slide
- Анимация кнопок: scale при hover
- Пружинная анимация иконок при появлении
- Плавная анимация прогресс-бара
