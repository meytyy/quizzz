# Быстрый старт

## Установка и запуск

```bash
# 1. Установить зависимости
npm install

# 2. Запустить dev-сервер
npm run dev

# Приложение откроется на http://localhost:5173
```

## Использование

1. **При первом запуске** вы увидите экран приветствия
2. **Нажмите "Загрузить flashcards_food_chem_kazakh.csv"** - файл автоматически загрузится из папки `public/`
3. **Или загрузите свой CSV файл** через drag & drop или кнопку выбора файла
4. **Начните тест** - выберите ответы, используя мышь или клавиатуру (цифры 1-9)
5. **Завершите тест** - нажмите "Завершить" после ответа на все вопросы
6. **Посмотрите результаты** - статистика, детализация, возможность экспорта

## Формат CSV

Приложение автоматически определяет формат CSV. Поддерживаются:

- **Стандартный**: `question, option1, option2, option3, option4, answer`
- **С Options**: `Question, Options, Correct_Letter, Correct_Answer` (ваш формат)
- **Альтернативный**: `question, correct, wrong1, wrong2, wrong3`
- **Простой**: `question, answer` (автогенерация вариантов)

## Структура проекта

- `src/components/` - React компоненты
- `src/utils/` - утилиты (парсинг CSV, хранение, форматирование времени)
- `public/` - статические файлы (CSV, иконки)
- `scripts/` - скрипты для сборки

## Проблемы?

Если CSV файл не загружается автоматически:
1. Убедитесь, что файл находится в папке `public/`
2. Или используйте функцию загрузки файла в приложении
