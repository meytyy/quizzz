# Quiz Flashcards App

Полностью рабочий одностраничный веб-приложение Quiz/Flashcards на React + Vite + Tailwind CSS для интерактивного тестирования с вопросами из CSV файла.

## 🚀 Возможности

- ✅ Парсинг CSV файлов с поддержкой разных форматов
- ✅ Интерактивный тест с подсветкой правильных/неправильных ответов
- ✅ Анимации и плавные переходы (Framer Motion)
- ✅ Статистика и результаты с детализацией
- ✅ Экспорт/импорт CSV файлов
- ✅ Редактирование вопросов в админ-панели
- ✅ Светлая и темная тема
- ✅ Адаптивный дизайн (мобильный и десктоп)
- ✅ Accessibility (поддержка клавиатуры, aria-атрибуты)
- ✅ Сохранение прогресса в localStorage
- ✅ Таймер и статистика времени

## 📝 Форматы CSV

Приложение поддерживает несколько форматов CSV:

### 1. Стандартный формат (рекомендуемый)
```csv
question,option1,option2,option3,option4,answer
Вопрос 1?,Вариант 1,Вариант 2,Вариант 3,Вариант 4,2
```
Где `answer` - это индекс правильного варианта (1-based) или текст.

### 2. Формат с Options
```csv
Question,Options,Correct_Letter,Correct_Answer
Вопрос 1?,"A) Вариант 1 | B) Вариант 2 | C) Вариант 3 | D) Вариант 4",B,Вариант 2
```

### 3. Альтернативный формат
```csv
question,correct,wrong1,wrong2,wrong3
Вопрос 1?,Правильный ответ,Неправильный 1,Неправильный 2,Неправильный 3
```

### 4. Простой формат (2 колонки)
```csv
question,answer
Вопрос 1?,Правильный ответ
```
Приложение автоматически сгенерирует 3 случайных варианта для создания MCQ.

### Дополнительные колонки

- `explanation` - объяснение правильного ответа (показывается после выбора)
- `tags` - теги для категоризации (через запятую)

## 🎮 Использование

### Загрузка CSV

1. При первом запуске вы увидите экран приветствия
2. Нажмите "Загрузить flashcards_food_chem_kazakh.csv" для загрузки локального файла
3. Или перетащите CSV файл в область загрузки
4. Или выберите файл через кнопку "Загрузить файл"

### Прохождение теста

- Выберите ответ, нажав на вариант или используя цифры 1-9
- Правильный ответ подсветится зеленым, неправильный - красным
- После выбора появится объяснение (если есть)
- Используйте стрелки ← → для навигации между вопросами
- Нажмите "Завершить" после ответа на все вопросы

### Результаты

После завершения теста вы увидите:
- Общий процент правильных ответов
- Количество правильных/неправильных ответов
- Общее время и среднее время на вопрос
- Детализацию по каждому вопросу
- Возможность экспортировать результаты в CSV

### Админ-панель

В админ-панели можно:
- Загрузить новый CSV файл
- Экспортировать текущие вопросы в CSV
- Скачать шаблон CSV
- Редактировать вопросы (inline)
- Добавить новые вопросы
- Удалить вопросы
- Сбросить все данные

## 🎨 Функции

### Accessibility

- Поддержка навигации клавиатурой:
  - `1-9` - выбор ответа
  - `←` / `→` - навигация между вопросами
  - `Enter` - завершить тест (на последнем вопросе)
- Aria-атрибуты для screen readers
- Фокусные состояния для всех интерактивных элементов

### Темная тема

Переключатель темы находится в правом верхнем углу. Предпочтение сохраняется в localStorage.

### Сохранение прогресса

- Прогресс автоматически сохраняется в localStorage
- При перезагрузке страницы можно продолжить с того же места
- Результаты также сохраняются

## 📦 Создание архива

Для создания ZIP архива с production сборкой и исходниками:

```bash
npm run build  # Сначала соберите проект
npm run zip    # Создайте архив
```

Будет создан файл `quiz_site.zip` в корне проекта, содержащий:
- `dist/` - production сборка
- Все исходники проекта
- CSV файл с вопросами
